 Global-Health-Trend-Analysis-Dashboard


 Overview
This Power BI dashboard analyzes global health outbreak trends, mortality rates, and country comparisons.

 Features
- Total Cases Analysis
- Total Deaths Tracking
- Mortality Rate Calculation
- Country Comparison
- Geographical Visualization
- Interactive Filters

 Tools Used
- Power BI
- DAX
- Power Query

 Dashboard Preview

 Insights
- Sierra Leone recorded the highest outbreak cases.
- Outbreaks peaked between 2014 and 2015.
- West African countries were the most impacted.
